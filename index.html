Probando hacer distintas cosas con JS

Carga de Datos

Primera parte del curso, donde se pueden cargar Datos de Peliculas como si fuera una DB. Se guardan en Storage para Practicar y permite visualizar lo guardado

Se usan: Arrays - Objetos - LocalStorage - Jquery - DOM - Animaciones

Consulta de Datos

Se realiza consulta a API con informacion de peliculas / series / juegos. Devuelve aquellas que se ajustan a la busqueda y visualiza rating de mas de un lugar. Se guarda informacion en LocalStorage para escalar otras funcionalidades.

Se usan: Arrays - Objetos - LocalStorage - Jquery - APIs - AJAX

Gestion DB

Seccion para consultar lo que se guardo en Storage, simulando DB y borrarlo (total o parcialmente)

Se usan: Arrays - Objetos - LocalStorage
